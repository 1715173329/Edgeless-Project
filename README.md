## 许可证
Edgeless的自编代码基于[MPL2.0](https://www.mozilla.org/en-US/MPL/)许可证开源，除了`setTheme.cmd`

## 如何使用
Clone此仓库，然后将Core部分的文件夹覆盖复制到您的PE项目文件夹中

当然，您的PE需要添加PECMD支持，我们推荐使用Wimbuilder构建您的PE核心
